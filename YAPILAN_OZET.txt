### 🎯 Roller & İzinler

**Super Admin**
- Tüm klinikleri yönetir
- Klinik admin oluşturur
- Sistem metriklerini görür
- Tüm verilere erişim

**Clinic Admin**
- Kendi kliniğini yönetir
- Müşteri/hayvan ekler
- Randevu onaylar
- Fiyat talebi yanıtlar
- Sağlık kaydı girer
- Davet kodu oluşturur

**Pet Owner**
- Kliniğe bağlanır (davet/QR)
- Hayvanlarını yönetir
- Randevu talep eder
- Fiyat bilgisi ister
- Hatırlatmaları görür
- WhatsApp deep link kullanır

### 🚀 Kurulum (5 adım)

1. Supabase projesi oluştur (https://supabase.com)
2. SQL migration çalıştır (001_initial_schema.sql)
3. Seed data ekle (seed.sql)

Auth: Supabase Auth (JWT)
Storage: Supabase Storage
Hosting: Vercel (Frontend), Supabase (Backend)
